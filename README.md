# AI Resume–JD Matching Tool

- An AI-powered ATS Resume Analyzer.

## Features

- Resume Upload
- Job Description Upload
- Skill Extraction
- Semantic Similarity
- AI Match Score
- Gemini AI Explanation
- Dashboard
- PDF Report

## Tech Stack

- Python
- Streamlit
- Sentence Transformers
- Gemini API
- Plotly
- ReportLab

## Installation

- pip install -r requirements.txt

- The project uses Google's supported `google-genai` SDK.

- streamlit run app.py
        or
- venv\Scripts\activate.bat python -m streamlit run app.py
- python -m streamlit run app.py
